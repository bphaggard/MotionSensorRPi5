# PIR sensor with LED, Camera and Email alert

Used components:
- Raspberry Pi 5
- PIR sensor HC-SR501
- RGB LED
- 220R resistors
- GPIO Extension Board
- 40 Pin GPIO Cable
- Camera Module
- Breadboard
- Jump wires
